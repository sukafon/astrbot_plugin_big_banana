# Changelog

本文件记录 Big Banana V2 的版本更新。V2 以新配置与新数据结构为基线，不维护 V1 配置或运行数据的兼容迁移。

## [v0.2.0] - 2026-07-12

### 兼容性变更

- 重构插件配置、生成流程和模块结构，V1 配置不再作为兼容基线；从旧版升级时建议重新生成并检查插件配置。
- 提供商配置改为模板列表，按 `capability` 区分图片与视频能力，并通过 `enabled_as_default` 和 `fallback_order` 配置默认回退顺序。
- 统一生成结果、参考图收集和提供商调度结构，插件内部旧模块路径不再兼容。

### 新增

- 新增 `save_images.r2_save`，支持在普通发图模式下将生成图片独立保存到 R2 图床，并与 URL 模式复用同一次上传。
- 新增智谱异步视频提供商，支持 CogVideoX-Flash 文生视频和单图生视频。
- 新增 `bnv` 视频预设和 `banana_video_generation` LLM 函数调用工具。
- 新增 OpenAI Responses、MiniMax Images、SiliconFlow Images 等提供商模板。
- 新增命令与 LLM 工具后台任务，可选择事件消息或主动消息发送结果。
- 新增上游插件后台回调，并向回调传递统一消息链、生成参数、会话来源及成功状态。
- 新增副脑提示词优化，可分别控制命令调用和 LLM 工具调用。
- 新增 Cloudflare Worker + R2 图床适配，使仅返回 base64 的提供商支持 URL 模式。
- 新增配置仪表盘，可管理常用配置、提供商、预设、头像替换及 Bot 人设参考图。
- 新增参考图隐私元数据清理。
- 新增全局与提供商级 `max_images` 限制。
- 新增群组冷却、头像编号说明、图片本地保存和无图片时文本响应等偏好配置。

### 改进

- 重构为命令处理、生成管线、提供商、客户端、守卫、配置和 schema 等独立模块。
- 改进多来源参考图收集、头像补充、引用消息处理和收集模式。
- 改进预设参数解析，增加整数、浮点数、布尔值及取值范围校验。
- 改进 LLM 工具描述、参数覆盖逻辑、内网图片 URL 防护和结果处理。
- 改进多 Key 重试、智能重试、提供商降级和错误信息。
- 支持 OpenAI Images 流式预览、自动尺寸推断及审核级别配置。
- 生成任务完成后才计算群组冷却时间。

### 修复

- 修复部分 AstrBot 版本中直接触发表情包返回字典时的兼容问题。
- 修复提供商配置在类型检查和 Web 配置接口中的类型错误。
- 在默认配置和插件内部预设中同时提供 `llm_default`、`llm_video_default`，配置中的同名预设可覆盖内部值。
